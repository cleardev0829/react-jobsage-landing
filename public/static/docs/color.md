```js
const PRIMARY = {
  lighter: '#C8FACD',
  light: '#5BE584',
  main: '#00AB55',
  dark: '#007B55',
  darker: '#005249'
};
const SECONDARY = {...};
const INFO = { ... };
const SUCCESS = {...};
const WARNING = {...};
const ERROR = {...};
...
```

You can pick colors here:

**Material color tool**

[https://next.material-ui.com/customization/color/](https://next.material-ui.com/customization/color/)

**Eva color tool**

[https://colors.eva.design/](https://colors.eva.design/)

We recommend picking colors with these values:

- Lighter : 100
- Light : 300
- Main : 500
- Dark : 700
- Darker : 900
